##### link和import区别

| link                         | import                                           |
| ---------------------------- | ------------------------------------------------ |
| 页面被加载，link会同时被加载 | @import引用的css会等到页面被加载完成之后再加载。 |
| 只适用与2.1之后的版本        | link是没有任何兼容问题的。                       |
| 支持使用js去控制dom改变样式  | 不支持                                           |
|                              | 只能加载css                                      |
